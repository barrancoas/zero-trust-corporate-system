## Definition of Done

A task is considered done when all of the following conditions are met:

- The task is fully implemented or written according to its description.
- The result is committed to the GitHub repository in the corresponding folder.
- The task is marked as done in the ProofHub board.
- All generated documentation follows the project standard: Markdown format, written in technical English.
